# Get started
1. Make sure you have installed Visual Studio 2015 Update 3
2. THis project is using SQL Server, make sure you have at least sql server express installed
3. Create new database e.g "sample" - for this project I've named it "sample"
4. Open Sample.sln
5. Set the Sample.DatabaseMigration as startup project
6. Run the project to execute the migration scripts and create the necessary table (including initial data)
